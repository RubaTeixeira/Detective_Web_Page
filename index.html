Detective Chatbot

Sobre esse chatbot

Torne-se o detetive e mergulhe em um intrigante caso de assassinato, inspirado nos clássicos contos de Sherlock Holmes.

Prepare-se para coletar pistas, analisar evidências e decifrar enigmas para solucionar o crime.

Baseado na série de jogos de cartas Elementar da Grok Games.


Resumo do Projeto

O jogo foi desenvolvido com a robustez do Dialogflow ES para processamento de linguagem natural, através um fluxo conversacional determinístico, justamente por se tratar de um jogo, pois existe a necessidade de se manter o contexto.

Para os processos de backend, foi utilizado o Google Cloud Run, que inclui os serviços de webhook para o funcionamento do jogo.

Ele executa dois processos, um para gerar a pistas aleatóriamente e outro para avaliar as respostas do quiz.
E para o armazenamento de dados e imagens do jogo utiliza o Google Storage.
